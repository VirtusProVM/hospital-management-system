# Hospital Management System

A comprehensive Java/Spring Boot web app for managing patients, doctors, appointments, and billing.

## Features

- Patient registration and management
- Doctor and specialty directory
- Appointment scheduling
- Billing and payments
- Secure authentication (Spring Security)
- Responsive UI (Bootstrap + Thymeleaf)

## Tech Stack

- Java 21
- Spring Boot 3.x
- MySQL
- Thymeleaf, Bootstrap

## Prerequisites
- [Maven](https://maven.apache.org/)
